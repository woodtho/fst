# 01 — Curriculum Map

The curriculum preserves the PFL2 program's **40 communicative training objectives**
(*objectifs de formation*) as the top-level learning modules. Each objective is the unit
of progress, mastery, and adaptation. Objectives are tagged along five cross-cutting axes
so the same content is reachable from any entry point the learner chooses.

## Taxonomy (the five axes)

1. **Level** — `A` (objectives A01–A18) or `B` (objectives B19–B40).
2. **Skill** — `reading`, `writing`, `listening`, `speaking`, `grammar`, `vocabulary`.
   Every objective has a *primary* skill and *secondary* skills it exercises.
3. **Theme** — communicative domain (personal info, workplace, meetings, telephone,
   government terminology, travel, daily life, scheduling, descriptions, instructions,
   problem solving, public service).
4. **Grammar concept** — the structures the objective introduces or consolidates
   (present tense, passé composé, imparfait, futur, conditionnel, subjonctif, pronouns,
   prepositions, adjective agreement, negation, question formation, comparatives, si-clauses,
   reported speech).
5. **Vocabulary domain** — workplace, government, meetings, administration, travel,
   personal information, time & scheduling.

A learner can filter by **any axis** (Level → Theme → Grammar concept → Skill →
Objective). The same module appears under every axis it is tagged with.

## Module anatomy (applies to all 40)

Each objective expands into five stages, gated by mastery (see
[05-adaptive-learning-framework.md](05-adaptive-learning-framework.md)):

| Stage | Contents | Exit condition |
|-------|----------|----------------|
| **Learn** | Concept explanation · vocabulary list · grammar notes · pronunciation guidance · example dialogues · example texts | Viewed (no gate) |
| **Practice** | Guided exercises, fill-in-blank, matching, sentence building, translation, error correction | ≥70% on guided set |
| **Consolidation** | Mixed review, scenario tasks, real workplace tasks, integrated R/W/L | ≥75%, ≥2 task types |
| **Self-test** | Timed + untimed quizzes, adaptive set, progressive difficulty | Attempted |
| **Mastery check** | Pass/fail recommendation, confidence score, weak-area report | Mastery ≥ 0.8 (see framework) |

## CEFR / GoC anchoring (informative)

GoC Level A ≈ CEFR A1–A2 functional range; GoC Level B ≈ CEFR B1. SLE levels map to
GoC **A / B / C**; this platform targets A and B and the SLE prep that precedes them.
Anchoring is used only to calibrate item difficulty, not shown as a hard equivalence.

## Skill coverage matrix

Every objective lists a **primary** skill (table above) and exercises **secondary**
skills. The platform guarantees each of the six skills is the *primary* focus of at least
four objectives, so a learner filtering by skill always has a full track:

| Skill | Primary in objectives |
|-------|-----------------------|
| Speaking | A01, A02, A03, A05, A08, A11, A12, A14, A16, B19, B20, B21, B22, B29, B37, B38, B40 |
| Listening | A06, A07, A09, A17, B26 |
| Reading | A10, A13, B28, B35 |
| Writing | A04, A15, A18, B23, B24, B25, B27, B30, B31, B32, B33, B34, B36, B39 |
| Grammar | cross-cutting — surfaced via "grammar concept" filter (see below) |
| Vocabulary | cross-cutting — surfaced via "vocab domain" filter (see below) |

> Grammar and Vocabulary are **cross-cutting tracks**: rather than owning objectives, they
> aggregate the grammar/vocab slices of all 40 objectives into focused study paths (e.g.
> "Passé composé" pulls drills from A11, B23, B33). This is how requirement §6
> (topic-focused study by grammar concept / vocab domain) is satisfied without duplicating
> content.

## Learner journey through the map

```
Level A ─┬─ A01 ─ A02 ─ A03 ... A18 ─┐
         │   (mastery-gated)         │
         └──► Level A readiness ──────┘
                       │
Level B ─┬─ B19 ─ B20 ... B40 ─┐
         │                     │
         └──► Level B readiness ┘
                       │
                       └──► SLE prep mode (Reading / Written Expression / Oral)
```

Objectives are *recommended* in numeric order but not hard-locked: a learner may jump
via any filter. The recommender (framework §6) nudges toward the next objective with the
highest expected learning gain given current mastery.
